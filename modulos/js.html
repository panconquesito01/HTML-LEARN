¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear contenido dinámico e interactivo en la web. Se ejecuta en el navegador (o en el servidor con Node.js) y puede modificar la estructura de la página, responder a eventos y comunicarse con servidores.

Sintaxis básica

// Comentario de una línea
/*
  Comentario
  de varias líneas
*/
let saludo = '¡Hola, mundo!';
console.log(saludo);

Usamos let o const para declarar variables. console.log() imprime en la consola del navegador.

Variables y Tipos

const nombre = 'Ana';      // String
let edad = 16;                 // Number
let activo = true;             // Boolean
let datos = {               // Objeto
  curso: 'Décimo grado',
  modulo: 'JavaScript'
};
let lista = [1, 2, 3];         // Array

JavaScript es de tipado dinámico: las variables pueden cambiar de tipo durante la ejecución.

Funciones

// Declaración clásica
function suma(a, b) {
  return a + b;
}

// Arrow function (ES6)
const resta = (a, b) => a - b;

console.log(suma(5, 3), resta(5, 3));

Las funciones encapsulan lógica reutilizable. Las arrow functions omiten la palabra clave function.

Control de flujo

// Condicionales
if (edad >= 18) {
  console.log('Adulto');
} else {
  console.log('Menor de edad');
}

// Bucles
for (let i = 0; i < 5; i++) {
  console.log(i);
}

let j = 0;
while (j < 3) {
  console.log(j);
  j++;
}

Manipulación del DOM

// Seleccionar elementos
const titulo = document.querySelector('h1');
const items = document.getElementsByClassName('item');

// Cambiar texto y estilo
titulo.textContent = 'Nuevo título';
titulo.style.color = 'var(--clr-accent)';

Eventos

const boton = document.querySelector('#miBoton');
boton.addEventListener('click', () => {
  alert('¡Has hecho clic!');
});

Con addEventListener escuchamos acciones del usuario (clic, teclado, scroll…) y ejecutamos código.

Asincronía y Fetch

// Fetch API (promesas)
fetch('https://api.example.com/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Async/Await (ES8)
async function cargarDatos() {
  try {
    const res = await fetch('https://api.example.com/data');
    const datos = await res.json();
    console.log(datos);
  } catch (error) {
    console.error(error);
  }
}
cargarDatos();

Módulos ES6

// archivo util.js
export function saludar(nombre) {
  return `Hola ${nombre}`;
}

// archivo main.js
import { saludar } from './util.js';
console.log(saludar('María'));

Con export y import dividimos el código en partes reutilizables.

Buenas prácticas

  • Usar const por defecto y let sólo si cambian.
  • Nombrar variables y funciones de forma descriptiva.
  • Evitar código global excesivo.
  • Comentar y mantener formato consistente (Prettier).
