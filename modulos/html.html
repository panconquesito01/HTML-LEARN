¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado que usamos para crear la estructura de las páginas web. Define cómo organizamos texto, imágenes, enlaces y otros recursos. Gracias a HTML5 podemos integrar audio, video, gráficos vectoriales (SVG) y más, sin necesidad de plugins externos.

Estructura básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi página</title>
  </head>
  <body>
    <!-- Contenido visible -->
  </body>
</html>

<!DOCTYPE html> activa el modo estándar. El atributo lang="es" en <html> mejora accesibilidad y SEO.

Etiquetas básicas

  • <h1>…</h1> hasta <h6>…</h6>: títulos jerárquicos.
  • <p>…</p>: párrafos de texto.
  • <br>: salto de línea.
  • <strong>…</strong>: énfasis fuerte.
  • <em>…</em>: énfasis.
  • <div>…</div> y <span>…</span>: contenedores genéricos.
<h1>Bienvenidos</h1>
<p>Este es un <strong>ejemplo</strong> de HTML</p>

Semántica avanzada

HTML5 añade etiquetas que describen mejor el propósito del contenido:

  • <header> y <footer>: encabezado y pie de sección.
  • <nav>: navegación.
  • <main>: contenido principal.
  • <article>: contenido independiente.
  • <section>: agrupaciones lógicas.
  • <aside>: contenido secundario.
  • <figure> y <figcaption>: medios con leyenda.
<figure>
  <img src="diagram.png" alt="Diagrama semántico">
  <figcaption>Diagrama de etiquetas</figcaption>
</figure>

Multimedia

Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Tu navegador no soporta audio.
</audio>

Video

<video controls width="640">
  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta video.
</video>

Canvas

<canvas id="miCanvas" width="200" height="100">
  Tu navegador no soporta canvas.
</canvas>

<script>
  const ctx = document.getElementById('miCanvas').getContext('2d');
  ctx.fillStyle = '#00aaff';
  ctx.fillRect(10, 10, 150, 75);
</script>

Formularios y validación

<form action="/enviar" method="post">
  <label>Nombre:
    <input type="text" name="nombre" required minlength="3">
  </label>
  <label>Email:
    <input type="email" name="email" required>
  </label>
  <button type="submit">Enviar</button>
</form>

IFrames y embebidos

Accesibilidad y ARIA

  • Usar alt en imágenes y label en formularios.
  • Atributos aria-label, role donde sea necesario.
  • Semántica adecuada para lectores de pantalla.
